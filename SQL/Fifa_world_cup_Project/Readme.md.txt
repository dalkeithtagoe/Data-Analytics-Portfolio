# FIFA World Cup Player Analysis (SQL)

## Project Overview

This project focuses on cleaning and analyzing a FIFA World Cup player dataset using SQL.

The dataset contains player information from multiple national teams and includes inconsistencies such as duplicate records, missing values, inconsistent units, and formatting errors. The primary objective was to transform the raw data into a clean dataset before performing exploratory data analysis.

---

## Business Objective

To demonstrate practical SQL skills by cleaning messy data and uncovering meaningful insights from football player statistics.

---

## Dataset Features

- Player Name
- Country
- Club
- Position
- Age
- Height
- Weight
- FIFA Rating
- Market Value
- Shirt Number
- FIFA World Ranking

---

## Data Cleaning

The following cleaning tasks were performed:

- Removed duplicate records
- Standardized text formatting
- Corrected inconsistent units
- Renamed columns
- Converted data types
- Handled missing values
- Removed unnecessary spaces

---

## Exploratory Data Analysis

Analysis included:

- Average FIFA Rating by Country
- Average Height by Country
- Market Value Comparisons
- Position Analysis
- Squad Age Distribution
- Country Rankings

---

## Key Findings

- National teams with higher FIFA ratings generally occupied stronger world rankings.
- Goalkeepers were typically the tallest players in the dataset.
- Midfielders represented the largest proportion of players.
- Market values varied significantly across countries, reflecting differences in squad strength.
- Cleaning the dataset greatly improved data consistency and analysis reliability.
