# Data Professionals Survey Dashboard (Power BI)

## Overview

This project analyzes survey responses from data professionals around the world using Power BI. The dashboard explores salary trends, work-life balance, job satisfaction, programming language preferences, and demographic insights.

## Objective

To transform survey data into an interactive dashboard that provides meaningful insights into careers within the data industry.

## Key Insights

- Salary varies across different data professions.
- Many respondents reported positive work-life balance.
- Python was one of the most popular programming languages.
- Survey participants represented multiple countries, highlighting the global demand for data professionals.
- Job satisfaction depends on more than just salary.
