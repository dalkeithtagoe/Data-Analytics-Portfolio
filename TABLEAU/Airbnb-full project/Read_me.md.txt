# Airbnb Listings Analysis (Tableau)

## Project Overview

This project explores Airbnb listing data using Tableau to uncover pricing trends, neighbourhood performance, and customer review patterns.

The dashboard enables users to interactively explore the Airbnb marketplace through dynamic visualizations.

---

## Business Objective

To understand how location, pricing, and customer reviews influence Airbnb listings and support data-driven decision-making.

---

## Dashboard Features

- Interactive Filters
- Dynamic Charts
- Geographic Analysis
- Price Comparisons
- Review Analysis

---

## Key Findings

- Listing prices vary considerably across neighbourhoods.
- Some neighbourhoods contain significantly more Airbnb listings than others.
- Review patterns provide useful insights into listing performance.
- Interactive filters allow users to compare locations and identify pricing trends efficiently.
